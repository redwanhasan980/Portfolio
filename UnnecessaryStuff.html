Reactive Racing Algorithm

2019 f1 tenth winning algorithm The Disparity Extender algorithm aim for the farthest available point. It processes sensor data to fill gaps between detected obstacles, making obstacle boundaries safer and more reliable. It extends or inflates obstacles in sparse or noisy data to prevent risky navigation through false gaps. In simulation, this means adjusting sensor readings to better represent obstacles for navigation.

3D Mapping

RTAB-Map (Real-Time Appearance-Based Mapping) is a SLAM algorithm that builds a 3D map by recognizing previously seen places using visual and depth data. It fuses RGB images and depth information from my depth camera to create a detailed, real-time 3D map of the environment.
